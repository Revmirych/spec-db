Описание задания
С помощью SQLAlchemy Core создайте подключение к PostgreSQL.

Реализуйте функции для базовых CRUD‑операций (создать, прочитать, обновить, удалить) для простой таблицы (например, «Пользователь» с полями id, name, email, age).

Добавьте обработку ошибок и откат (rollback) транзакции при возникновении исключения в каждом из методов кроме чтения.

Для проверки данных в БД можно использовать DBeaver.
