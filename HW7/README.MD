Критерии оценки
Задание считается выполненным, если:

прикреплена ссылка/файл с выполненным заданием;
в файле/по ссылке содержится выполненное задание;
доступ к материалам открыт.
Задание считается невыполненным, если:

файл/ссылка с заданием не прикреплены или отсутствует доступ;
файл/ссылка не содержат выполненное задание.

Напишите тесты для PostgreSQL БД, используя pytest, фикстуры и Faker для генерации данных. Тесты должны покрывать CRUD-функционал и реализовывать паттерн как позитивных, так и негативных тестов.
